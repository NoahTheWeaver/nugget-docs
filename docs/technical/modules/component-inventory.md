# Component Inventory

**Technical name:** `nugget_component_inventory`\
**Depends on:** `mrp`, `stock`, `nugget_variant_name`, `nugget_inventory_status`\
**Author:** Nugget Scientific

Provides a component-level inventory report that accounts for components mounted inside assemblies (BOMs). Standard Odoo stock reports only show standalone inventory — this module bridges the gap.

## Key Views

- **Component Inventory report** — Shows where components are consumed across assemblies
