# Purchase Configurator

**Technical name:** `nugget_purchase_configurator`\
**Depends on:** `purchase`, `stock`, `sale`\
**Author:** Nugget Scientific

Adds a product template selector to Purchase Order lines, mirroring the sale order configurator pattern. Allows selecting a product template first, then choosing specific variants — streamlining purchasing for products with many variants.

## Key Views

- **Purchase Order form** — Product template field on order lines

::: warning JS Components
This module includes OWL 2 JavaScript components. Check browser console for errors after updates.
:::
