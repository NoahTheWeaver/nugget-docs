## Upcoming Renewals and Expirations
How to see what's expiring soon. What to do when a contract is up for renewal.

When a subscription renews, Odoo may create a new sales order. You must re-link each covered system to the new subscription. If you skip this, dispatch will see covered systems as uncovered.
